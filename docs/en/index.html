About Noto Sans CJK

Noto Sans (Source Han Sans) is an open source sans serif font jointly developed by Google and Adobe. It has the features of pursuing beauty, seven weights, and the ability to modify and redistribute under the SIL Open Font License. It is also known as "Source Han Sans" or "Source Han Sans" under the Adobe brand.
Among the many languages, Japanese, traditional / simplified Chinese, and Korean, which have overwhelmingly many character variations, are developed separately from other languages, and are finally released as the "Noto Sans CJK" series. The Japanese version is "Noto Sans CJK JP".

About Noto Sans CJK Version 2.004

The latest version of Noto Sans CJK, version 2.004, was released in April 2021.
Noto Sans CJK has been continuously updated since its initial release in 2014, with many improvements such as support for Hong Kong, the addition of the ligature "㋿", and support for variable fonts. The latest version as of March 2022 is 2.004. The details of the updates, including each version, are described in the following pdf. (This pdf itself is for Source Han Gothic, which is the Adobe version of Noto Sans CJK, but it also applies to Noto Sans CJK.)

About Noto Sans CJK JP min

Noto Sans CJK has a huge number of Japanese characters (mainly kanji), but the file size has increased significantly as a result. This is especially problematic when using web fonts, as it can have a negative impact on page loading.
One approach that is often taken is to subset the font.
Noto Sans CJK JP min has been significantly reduced in size by extracting JIS first level kanji and part of JIS second level kanji, which is not a problem for normal use.
It also supports the latest version 2.004 as of March 2021.
